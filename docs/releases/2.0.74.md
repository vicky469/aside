# Release Highlights

This release makes it easier to detach a side note from text without deleting the note itself.

## Highlights

- Selecting text that already has an anchored side note now changes the editor menu action to `Remove anchor from side note`.
- Removing the anchor keeps the note content intact and marks the thread as orphaned instead of creating a duplicate side note.

## Notes

- There are no note-format or vault-data migrations in this release.
- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
