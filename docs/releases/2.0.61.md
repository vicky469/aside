# Release Highlights

This release updates the bundled `aside` Codex skill for structured page-note workflows.

## Highlights

- The bundled `aside` skill now treats "one point a note/comment" as one parent Aside thread with each point appended as a child comment, instead of creating separate page-note threads.
- Adds a repo helper for creating a page-note thread with ordered child comments, which keeps bulk note import and repair workflows aligned with Aside's threaded storage model.
- Helper-created side notes now refresh both note-path and source-id sidecar storage, so renamed-note recovery and synced source identities stay aligned.

## Notes

- Users who already installed the external `aside` Codex skill will get the updated skill content automatically when this plugin release starts on desktop.
- Aside still does not silently install a new external Codex skill for users who have not opted into that workflow. Built-in `@codex` side notes, including desktop local runtime use, do not require a separate skill install.
- There are no note-format or vault-data migrations in this release.
