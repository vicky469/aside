# Release Highlights

This release removes the dev-only diagnostics blocker from the mobile compatibility path.

## Highlights

- Removed the top-level Node `path` dependency from plugin startup so Aside can load without forcing a desktop-only manifest.
- Kept the local log inspector and `Locate log` behavior runtime-gated to local desktop environments instead of blocking mobile installs.
- Updated the submission checklist to reflect the real remaining task: a manual Android or iOS smoke test before the marketplace PR.
