# Release Highlights

This release tightens Aside's Codex skill handling and makes deleted-note comment cleanup more durable across synced state.

## Highlights

- Aside refreshes an already installed Codex skill on desktop startup, but no longer creates the Aside skill from a legacy SideNote2 skill.
- Deleted markdown files now write a synced delete tombstone when Aside has known comments from sidecars, open state, aggregate state, snapshots, or pending sync events.
- Added regression coverage for deleting a note whose comments only exist in synced plugin-data snapshots.

## Notes

- Install the Aside Codex skill separately when using external headless workflows. Aside keeps an installed skill current, but does not install or remove it for you.
