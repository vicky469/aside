# 2.0.85

## What's new

- **Plugin-owned anchored annotations**: Agent annotation requests such as `@codex 加批注` can now return exact source-text proposals that Aside turns into real selection-anchored side notes. Users no longer need helper scripts in the vault for this flow.

## Fixed

- Soft-deleted side note cards can be clicked in deleted mode so their restore and permanent-delete actions become visible.
- Restore and permanent-delete actions now appear for deleted cards even when shown from index/source views, including orphaned anchored notes.
- Regenerating agent replies preserves the original output card and keeps retry state attached to the right thread.
- Deleted agent output cards remain discoverable through the trash view instead of being filtered out of stored comment sync.

## Notes

- Annotation creation still depends on the agent quoting exact source text from the current markdown note. If the source text no longer matches, Aside keeps the visible reply and logs the unmatched proposal instead of creating a wrong anchor.
