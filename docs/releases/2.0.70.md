# Release Highlights

This release makes the generated Aside index calmer and more reliable, especially while switching files, deleting folders, and loading comments from synced storage.

## Highlights

- `Aside index.md` now ignores empty-space clicks in generated index rows and in the blank preview surface, preventing the full-page flash caused by Obsidian preview rerenders.
- The selected file in the index sidebar now follows the current file immediately instead of lagging behind the previous selection.
- Startup index hydration now uses persisted side-note cache and synced snapshots directly, avoiding unnecessary source-note reads.
- Deleted folders no longer surface missing sidecar cache files as uncaught `ENOENT` errors.
- Legacy inline `<!-- Aside comments -->` migration code is isolated from the normal storage path, while old inline blocks and legacy SideNote2 links remain import-compatible.

## Notes

- No manual migration is required. Old inline comment blocks are still imported and stripped automatically when encountered.
- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
