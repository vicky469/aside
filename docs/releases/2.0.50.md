# Release Highlights

This release tightens the draft-save flow for anchored notes, simplifies more of the sidebar chrome, and makes side-note navigation and note-link creation more reliable.

## Highlights

- Empty anchored side notes can now be added directly from the sidebar and collapse back to preview after save, while empty page-note drafts stay blocked until you enter content.
- Sidebar cards are simpler and more stable: pin state now persists, bookmark support has been removed, the preview expand affordance is visible again, and the meta/action layout is cleaner.
- Opening side-note deep links is more reliable, and creating linked notes from comments now sanitizes unsafe or invalid paths before file creation.

## Notes

- This release removes bookmark-specific sidebar behavior that was previously exposed in the UI.
- No Aside thread-format migration is required.
