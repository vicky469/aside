# Release Highlights

This release makes drafting side notes feel more like writing normal markdown, while tightening source-note navigation from the index sidebar.

## Highlights

- Rich HTML paste in side-note drafts now converts through Obsidian's markdown converter instead of dropping useful structure from copied content.
- Draft editors now continue markdown lists when pressing Enter inside list items.
- Revealing page notes from the index sidebar preserves the source context instead of forcing an unnecessary scroll to the top of the note.
- The sidebar keeps its last supported note context when focus briefly moves to unsupported files.

## Notes

- There are no note-format or vault-data migrations in this release.
- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
