# Release Highlights

This release adds note-scoped markdown export and tightens the note sidebar interactions so export and thread controls stay out of the way until you need them.

## Highlights

- Added a note-sidebar export action that writes the current file's side notes to `SideNote2/exports` as clean markdown.
- Exported markdown keeps anchored selections as thread labels, preserves connected side-note references as wiki links, and strips noisy `Mentioned:` / side-note URI blocks.
- Export links now use alias-style wiki paths when needed, so duplicate file names stay unambiguous while the visible link text stays minimal.
- The note sidebar toolbar is cleaner: `Thought Trail` hides the search and add-note row, and nested thread toggles appear only after you single-click a comment card.

## Notes

- Exports are stored as flat files under `SideNote2/exports`, and re-exporting updates the existing file for that source note.
- No note-format or vault-data migration is required for this release.
