# Release Highlights

This release rebuilds the index sidebar around file-level navigation and hardens Thought Trail rendering.

## Highlights

- Simplifies `Aside index.md` to a folder/file list while keeping the header image and caption styling stable.
- Lets index file links filter the Aside sidebar instead of jumping directly to source notes.
- Adds index sidebar List and Thought Trail modes with compact file filtering, resolved visibility, delete actions, and source-note redirects from thread cards.
- Keeps the selected index sidebar tab when changing file filters instead of always returning to List.
- Fixes duplicate sidebar rendering caused by overlapping async index renders.
- Fixes Thought Trail Mermaid rendering by removing custom Obsidian click links from direct Mermaid input and cleaning markdown links from edge labels.

## Notes

- Thought Trail nodes remain clickable through Aside's own DOM binding; edge labels are display text only and do not redirect.
- The release artifact check covers `main.js`, `manifest.json`, and `styles.css` for source maps, embedded sources, local paths, and common secret patterns.
