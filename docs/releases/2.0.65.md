# Release Highlights

This release completes the public rename from SideNote2 to Aside and polishes the generated index selection state.

## Highlights

- The plugin now ships as `Aside` with plugin id `aside`, including renamed settings, commands, generated index paths, helper scripts, and Codex skill references.
- Legacy SideNote2 data paths and comment links remain recognized so existing side-note data can continue migrating into the current Aside storage flow.
- The selected file in `Aside index.md` now uses a tighter inline accent highlight instead of a full row highlight with a left strip.

## Notes

- Because the plugin id changed from `side-note2` to `aside`, this release installs into `.obsidian/plugins/aside`.
- The shipped plugin assets are still `main.js`, `manifest.json`, and `styles.css`.
