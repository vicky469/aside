# 2.0.84

## What's new

- **Cleaner note mode toolbar**: Pin, deleted, and add-page-note toolbar actions now appear only in the note sidebar's List tab. Todo, Agent, Tags, and Thought Trail modes keep their mode controls without showing irrelevant file actions.

## Fixed

- Replaced the draft editor's direct `InputEvent` check with Obsidian-safe cross-window type checking.
- Removed redundant TypeScript assertions reported by Obsidian's plugin quality checks.

## Notes

- The Tags tab label stays plural because it represents the tag browsing/filtering view, not a single tag.
