# 2.0.92

## What's new

- Make Aside settings searchable in Obsidian 1.13 and later while preserving the legacy settings interface for Obsidian 1.12.7.
- Add optional authenticated remote cache purging after unpublishing or republishing a public note, with credentials stored through Obsidian SecretStorage.
- Keep link suggestions, tag suggestions, move targets, Thought Trail, copy, paste, and share workflows responsive through centralized vault indexing and explicit clipboard actions.

## Notes

- Existing publishing behavior is unchanged unless **Remote cache purge** is enabled and configured.
- Release builds now run the shared compliance gate on Node.js 20, 22, and 24 and publish immutable, verified artifacts without embedded source maps or source content.
