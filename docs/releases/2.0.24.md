# Release Highlights

This release tightens the comment lifecycle in the sidebar and removes stale deleted-file threads from the index view.

## Highlights

- Added page-sidebar soft delete with restore, deleted-only filter, and retention-based cleanup.
- Tightened agent reply regeneration so retries stay attached to the same reply card instead of creating duplicate cards.
- Fixed the index sidebar so comments from deleted source files no longer linger after the source note is removed.

## Notes

- The deleted-comments flow is page-sidebar only. The index sidebar continues to hide deleted threads from the aggregate view.
