# Release Highlights

This release fixes a sidebar regression around nested comment toggles.

## Highlights

- Threads with nested replies now always keep their show or hide icon visible, even after you turn nested comments off from the top toolbar.
- The top nested-comments control now acts as the default for all threads instead of removing each thread's local toggle.
- Individual threads can still be expanded or collapsed after the global nested-comments control changes.

## Notes

- There are no note-format or vault-data migrations in this release.
