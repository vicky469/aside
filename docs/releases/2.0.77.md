# 2.0.77

## What's new

- **Nested anchored comments**: Anchored comments can now be moved under another thread while keeping their source selection highlights intact. Moving an anchor into a thread removes the duplicate top-level card instead of leaving the original behind.
- **Quieter workflow feedback**: Routine action notices are suppressed, including missing/orphan note warnings that are already represented visually in the sidebar.
- **Compact agent status**: The settings UI is slimmer, with the previous preferred runtime setting removed and agent status text reduced to the compact inline form.

## Fixes

- **Local HTML links open inside Obsidian**: `.html` and `.htm` links mentioned in comments now open as Obsidian tabs when the file exists in the vault, instead of launching the default browser.
- **Pinned file navigation stays pinned**: Clicking a comment card while the sidebar is pinned to a markdown file selects the card without switching the active editor file.
- **Nested selection highlights remain visible**: Moving selected-text comments into another thread no longer drops the purple source highlight or changes the anchor text rendering.
