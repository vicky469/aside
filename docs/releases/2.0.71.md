# Release Highlights

This release makes `Aside index.md` more useful and quieter to scan when comments include tags.

## Highlights

- `Aside index.md` now shows each file's side-note tags inline after the file name, so sidebar comment tags can be explored from the generated index.
- Index file rows now have a little more vertical spacing, making dense file lists easier to read.
- Regular index file names now use the theme's normal text color instead of the purple link color, while the selected file keeps its accent highlight.

## Notes

- There are no note-format or vault-data migrations in this release.
- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
