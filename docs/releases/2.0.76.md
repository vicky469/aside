# 2.0.76

## What's new

- **Todo tab default**: The index sidebar now opens on the Todo tab when `@todo` comments exist in the vault. The tab is disabled (not just faded) when there are no todos, and falls back to List automatically.
- **Agent tab removed from index sidebar**: The Agent tab is no longer shown in the index sidebar mode control.

## Fixes

- **Tag extraction**: Tags starting with digits (e.g. `#128`, `#42`) are no longer extracted. Matches Obsidian's rule that tags must start with a letter or underscore — prevents agent-generated text like "Build #128 / PR #42" from producing spurious tags in the index.
- **Todo tab interactive from cached list view**: Switching from Todo → List → Todo no longer gets stuck. The cached list render now computes live thread group counts so the Todo tab stays clickable.
- **Thought trail tab properly disabled**: The Thought Trail tab is now disabled when there are no actual connections to show (wikilinks in comments or shared-tag matches with other vault files). Previously it could appear enabled with an empty view.
- **Index sidebar scope**: All-comments index view now correctly scopes threads vault-wide when no file filter is selected, enabling Todo and Thought Trail to work without requiring a file selection first.
