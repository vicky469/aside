# Release Highlights

This release clarifies Aside empty-state guidance in the sidebar.

## Highlights

- The index sidebar empty state now points users to click a file in the index to filter the sidebar and inspect that file's side notes.
- The note sidebar empty state now explains both ways to start: use the add button for a page side note, or select text and right-click "Add comment to selection" for an anchored note.

## Notes

- There are no note-format or vault-data migrations in this release.
