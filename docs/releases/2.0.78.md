# 2.0.78

## What's new

- **More reliable agent replies**: Aside now persists agent replies more defensively, including streamed replies and sidebar updates, so completed agent output is less likely to be lost or left in a transient state.
- **Clearer agent run context**: Agent cards can show run metadata, and the index opens more predictably when reviewing generated agent activity.
- **Claude runtime compatibility**: Claude runs now receive the expected allowed tool list, and skill usage metadata is extracted from actual tool calls instead of the session-wide skill list.

## Fixes

- **Empty Aside view opens correctly**: Opening Aside without a current markdown note now initializes the sidebar view instead of silently failing.
- **Pasted clipboard blobs stay compact**: Large pasted payloads and long URLs are shortened into cleaner comment text while preserving useful destinations.
- **Cleaner settings surface**: The extra runtime re-check control was removed from settings to keep the agent status area focused.

## Notes

- There are no note-format or vault-data migrations in this release.
