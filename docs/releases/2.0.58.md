# Release Highlights

This release hardens Aside sync across devices and makes the generated index easier to use.

## Highlights

- Preserves mobile and desktop replies under the same thread when synced plugin data arrives out of order or from compacted snapshots.
- Adds source-note identity recovery so renamed markdown files can keep their sidebars without stealing notes from same-name files in other folders.
- Speeds up visible-sidebar sync polling so mobile changes appear faster on desktop for the currently focused notes.
- Makes generated index file rows collapsible with smaller `h5` headings, cleaner file labels, and safer native fold-button handling.
- Fixes mobile card controls so tapping side-note cards stays inside the side-note panel instead of navigating the workspace unexpectedly.

## Notes

- Source sidebars now use stable source IDs with path-hash fallback for older cache data.
- Ambiguous legacy recoveries are intentionally skipped instead of mutating the wrong file's sidebar.
