# 2.0.81

## What's new

- **Narrow sidebars keep note actions visible**: The note toolbar now preserves the pin, deleted-notes, nested-comments, and page-note actions when the sidebar is squeezed.
- **Clearer deleted-note mode**: Deleted-note view now explains the seven-day retention window before permanent cleanup.
- **Safer toolbar modes**: Page-note and filter actions are disabled while an exclusive pinned or deleted view is active, reducing accidental context switches.

## Notes

- No vault-data migration is required for this release.
