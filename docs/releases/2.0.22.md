# Release Highlights

This release adds built-in `@codex` side notes on desktop Obsidian. Save a side note with `@codex`, watch the reply stream in the sidebar, and review agent threads from a dedicated Agent tab without needing the external Aside CLI or skill for the built-in flow.

## Highlights

- Added built-in `@codex` dispatch for side notes on desktop Obsidian, with Aside appending the final reply back into the thread.
- Added streaming run state, persisted agent history, retry support, and a dedicated Agent tab with success and failure filters in the index sidebar.
- Simplified onboarding around built-in `@codex`, including a read-only Codex status check in settings and updated docs and product copy.

## Notes

- Built-in `@codex` requires desktop Obsidian with a filesystem-backed vault and a working local Codex installation.
- External share-link handoff still exists for advanced workflows, but built-in `@codex` is now the primary path.
