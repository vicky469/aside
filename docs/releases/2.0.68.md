# Release Highlights

This release addresses Obsidian CSS lint warnings from the 2.0.67 release assets.

## Highlights

- Removed broad `!important` overrides from the shipped stylesheet.
- Removed selectors and properties flagged by Obsidian CSS lint, including `:has`, `all`, `column-gap`, `text-decoration-thickness`, `text-decoration-color`, and `mjx-container` type selectors.
- Updated CSS style checks to keep the toolbar and index styles covered without relying on forbidden `!important` rules.
- Updated release-facing README and version bump automation to use marketplace release wording instead of pre-marketplace wording.

## Notes

- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
