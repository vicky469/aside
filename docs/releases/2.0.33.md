# Release Highlights

This release makes threaded side-note editing feel lighter in the sidebar and adds finer control over long reply chains.

## Highlights

- Parent side-note cards can now hide or show only their own nested replies, so long threads are easier to collapse without changing the rest of the note.
- Thread entry edits now stay inline inside the existing card, with smoother save behavior and outside-click autosave for sidebar interactions.
- Sidebar action chrome stays more stable while hovering and editing, which reduces card jumpiness during reply management.

## Notes

- The settings tab no longer shows the built-in Codex status panel.
- There are no note-format or vault-data migrations in this release.
