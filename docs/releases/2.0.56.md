# Release Highlights

This release cleans up remaining legacy hidden Aside blocks and hardens plugin-data sync for Obsidian Sync users.

## Highlights

- Recognizes old object-shaped `<!-- Aside comments -->` blocks, migrates any missing thread data into canonical storage, and strips the block from source markdown.
- Re-runs storage and sync migrations so vaults that already passed earlier migrations still get cleaned up.
- Merges Aside sync state with the latest plugin data before writing, reducing the chance that two active synced devices overwrite each other's event logs.
- Keeps sidecar files as rebuildable local cache; synced plugin data remains the cross-device source when Obsidian Sync is enabled for plugin data.

## Notes

- Users who want cross-device Aside sync should enable Obsidian Sync for plugin data.
- Source markdown no longer needs hidden Aside data after migration.
