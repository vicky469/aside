# Release Highlights

This release makes note drafting feel calmer and faster by reducing the sidebar flash when opening a new side note, while also giving the note toolbar a darker sidedock-style surface.

## Highlights

- Opening a new anchored or page-note draft now skips the extra pre-focus sidebar rerender, so the textarea appears faster with less visual flash.
- Draft focus still lands the cursor in the textarea automatically, but avoids the temporary active-highlight treatment that made the toolbar and thread area feel jumpy.
- New note-sidebars use a darker sticky toolbar surface that better matches Obsidian's sidedock chrome.

## Notes

- Draft selection hashes are now materialized lazily at save time; no note data migration is required.
