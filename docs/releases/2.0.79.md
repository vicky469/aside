# 2.0.79

## What's new

- **Current Aside storage only**: Aside no longer reads or migrates old inline source-note comment blocks or old plugin sidecar paths during normal runtime. Current storage is the Aside plugin data file plus Aside sidecar caches.
- **Clearer storage documentation**: The storage docs now describe the synced `data.json` event state, local sidecar caches, and how device-to-device comment sync should be reasoned about.
- **Full-thread search results**: Sidebar search now shows the complete matching thread, so replies and surrounding context stay visible when a keyword matches.
- **Source-note wikilinks in Thought Trail**: Thought Trail now follows wikilinks and embeds from the source markdown notes themselves, not only links written inside Aside comments.
- **Sidebar polish**: Tag-selection checkboxes now use a flexible row layout, and filter chips avoid the promoted shadow styling.

## Notes

- If you are upgrading from a very old SideNote2 or early Aside build whose comments only exist in legacy inline blocks or old plugin cache files, first open the vault with `2.0.78` and let it migrate current comments, then upgrade to `2.0.79`.
- Users already on current Aside storage do not need to run a vault-data migration for this release.
