# Release Highlights

This release makes sidebar pinning more explicit and keeps the optional external `aside` Codex skill aligned automatically for users who already installed it.

## Highlights

- Pinned side notes no longer silently narrow the sidebar list on their own. Instead, Aside now keeps pinning as a saved selection and gives you an explicit pinned-only toggle when you want to focus on that subset.
- If you already installed the external `aside` Codex skill for handoff workflows, Aside now refreshes that installed skill automatically when a newer plugin release starts up.
- The bundled `aside` skill guidance now distinguishes normal CLI/chat explanations from in-thread Aside write-back flows, which prevents pasted thread URIs from being treated as automatic reply instructions.

## Notes

- There are no note-format or vault-data migrations in this release.
- The automatic skill refresh only applies when the external `aside` skill is already installed; built-in `@codex` side notes still do not require any separate skill install.
