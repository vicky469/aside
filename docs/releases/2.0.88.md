# 2.0.88

## What's new

- **Recognizable index filename**: Aside's generated vault-wide index is now named `🐰 Aside Index.md`.

## Migration

- Existing `Aside index.md` files are renamed automatically when the plugin updates.
- Open sidebar and draft state follows the renamed index without creating a second generated note.
- If `🐰 Aside Index.md` already exists, Aside keeps the legacy index active and shows a notice instead of overwriting the existing file.
- If renaming or saving the migrated setting fails, Aside rolls back to the legacy filename and leaves the existing index intact.

## Notes

- The index remains at the vault root; this release does not add filename or folder settings.
