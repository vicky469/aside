# Release Highlights

This release tightens Aside's release safety and rendering hardening as part of marketplace submission prep.

## Highlights

- Added an automated shipped-artifact inspection step that blocks source-map markers, embedded sources, obvious secrets, and local-only paths from public releases.
- Hardened comment rendering paths and added regression coverage so HTML-like input stays inert in the draft preview.
- Simplified the public GitHub bug report flow.
