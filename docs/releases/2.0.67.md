# Release Highlights

This release includes the pending local changes that were staged after 2.0.66, with runtime hardening for sidebar rendering, Thought Trail diagrams, and release artifacts.

## Highlights

- Sidebar, comment highlight, and agent stream rendering now use owner-document DOM checks so UI interactions are safer across Obsidian windows and embedded document contexts.
- Thought Trail Mermaid rendering now resolves either Obsidian's loaded Mermaid runtime or the active window runtime, with extra tests for renderer fallback behavior.
- The build no longer needs the `builtin-modules` package and now uses Node's built-in module list directly.
- Release automation now generates artifact attestations for `main.js`, `manifest.json`, and `styles.css`.

## Notes

- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
