# 2.0.95

## What's new

- Keep the public plugin source archive cleaner for automated Obsidian release scans.
- Move the reference cache-purge Worker source out of the public plugin repository while preserving the optional plugin-side broker client.
- Add release guards that reject accidental `fetch` usage in shipped plugin artifacts.
- Keep the public Aside build independent of private repositories or private package credentials.

## Notes

- No user migration is required.
- Release artifacts remain limited to `main.js`, `manifest.json`, and `styles.css`.
- Vault indexing and clipboard access remain intentional, user-facing capabilities documented in the README.
