# 2.0.83

## What's new

- **PDF page notes**: PDFs can now have page-level Aside threads in the sidebar and `Aside index.md`.
- **Current Aside storage**: PDF page notes use the same sidecar and sync-event storage as markdown side notes. Aside does not write comment data into PDFs or restore the old attachment-comment storage path.

## Fixed

- Unsupported non-PDF files now show the unsupported-file sidebar state instead of continuing to display the previous note's side notes.

## Notes

- PDF support is page-note only. Text-anchored notes, highlights, re-anchor actions, and selection-based comments remain markdown-only.
- No vault-data migration is required for this release.
