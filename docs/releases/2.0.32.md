# Release Highlights

This release makes threaded side-note conversations behave more predictably while keeping comment editing faster from the sidebar.

## Highlights

- Agent replies triggered from a child comment now land directly after that child instead of falling to the end of the thread.
- Persisted side-note cards can now enter edit mode with a double click, so revising a reply is faster from the sidebar.
- Double-click edit ignores deleted cards and interactive targets, so normal buttons and links keep their existing behavior.

## Notes

- The page-note toolbar still uses the top-right `Add page note` action.
- Child comment ordering in existing threads is preserved; only new generated replies now insert closer to the comment that triggered them.
