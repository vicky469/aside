# 2.0.80

## What's new

- **Selected-file index cards**: The index sidebar List now shows only the selected file's comment cards instead of pulling in comments from related files.
- **Cleaner file filter label**: The linked-file count is hidden in List mode because those related files are not shown as cards there.
- **Thought Trail keeps related files**: Thought Trail still uses the selected file as the root and can show related files in the graph.

## Notes

- No vault-data migration is required for this release.
