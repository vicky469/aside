# 2.0.86

## What's new

- **Sidebar tab visibility settings**: You can now turn the Todo and Agent sidebar tabs on or off from Aside settings. Both stay on by default so existing users still discover the workflows.
- **Narrower sidebar control**: Hidden Todo and Agent tabs are removed from both note sidebars and the generated index sidebar instead of taking up disabled toolbar space.

## Fixed

- Adding or revealing a PDF page note from the sidebar no longer shows a false Markdown view jump failure notice.

## Notes

- This does not disable Todo parsing, agent replies, stored threads, or existing comments. It only controls whether the dedicated sidebar tabs are shown.
- If a hidden Todo or Agent tab was the active sidebar mode, Aside falls back to `List`.
