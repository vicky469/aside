# Release Highlights

This release tightens how sidebar comment cards open so the index view stays local and predictable.

## Highlights

- Double clicking a sidebar comment card now opens the comment target instead of starting edit mode.
- In a normal note sidebar, double click still jumps to the source selection in the source note.
- In `Aside index.md`, double click now reveals the matching index-note preview instead of redirecting to the source file, since the explicit redirect icon already handles source jumps.

## Notes

- Single click on a comment card now just marks it active in the sidebar.
- There are no note-format or vault-data migrations in this release.
