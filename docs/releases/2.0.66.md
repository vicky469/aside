# Release Highlights

This release fixes Thought Trail tab rendering for linked-file scopes and removes a slow refresh path when switching sidebar tabs.

## Highlights

- Thought Trail now includes incoming linked files when the current note is only reached through another note's links.
- Switching between List, Tags, and Thought Trail reuses loaded sidebar data instead of reparsing the current note.
- Thought Trail availability and render diagnostics now include scoped file and graph counts to make future tab issues easier to verify.

## Notes

- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
