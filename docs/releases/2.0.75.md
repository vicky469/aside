# Release Highlights

This release adds Claude CLI support for the agent, introduces a sticky file pin in the sidebar, and includes a run metadata footer for agent threads.

## Highlights

- Agent threads now support the Claude CLI as a backend in addition to the existing options.
- A metadata footer is shown at the bottom of agent run entries so you can see run details at a glance.
- Sidebar panels now support a sticky file pin, letting you lock a specific file into view regardless of editor navigation.

## Notes

- There are no note-format or vault-data migrations in this release.
- The shipped plugin assets remain `main.js`, `manifest.json`, and `styles.css`.
