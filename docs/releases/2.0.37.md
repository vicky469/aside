# Release Highlights

This release makes the note sidebar easier to scan and easier to use as a lightweight bookmark surface.

## Highlights

- The note sidebar now supports both a list view and a thought-trail view rooted on the current file, so you can switch between flat comment scanning and local graph navigation without leaving the note.
- Bookmarked threads now have a first-class bookmark state, a bookmark filter in the note sidebar, and a consistent bookmark marker on saved cards and draft controls.
- Bookmarked notes can now save without body text, and active comment cards keep a stable accent frame instead of flashing through a background-color transition while you add or save notes.

## Notes

- There are no note-format or vault-data migrations in this release.
