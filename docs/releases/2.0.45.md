# Release Highlights

This release makes sidebar pinning durable again and cleans up the optional external Codex skill setup docs.

## Highlights

- Sidebar pins now persist per file in the saved Aside view state, so both the pin markers on comment cards and the pinned-only toolbar view come back when you return to that note.
- The README now treats the public `aside` Codex skill as optional advanced setup and points external Codex users at the GitHub `$skill-installer` flow instead of implying a separate local Aside CLI install.
- The remote bridge base URL setting copy now explicitly allows either HTTP or HTTPS for localhost and private LAN development.

## Notes

- Pin state is persisted as per-user sidebar workspace state, not inside the markdown note itself. Bookmarks remain the note-backed long-lived state.
