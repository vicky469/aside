# Release Highlights

This release fixes wiki-link handling for nested Aside thread replies so linked notes stay connected even when the link lives in an older child comment instead of the latest entry.

## Highlights

- Fixed derived comment links so Aside now scans every entry in a thread when building wiki-link metadata and backlinks.
- Fixed the index file filter graph and Thought Trail view so thread connections from child replies are preserved instead of disappearing after newer replies are added.
- Added regression coverage for derived-link metadata, index graph construction, and Thought Trail rendering for multi-entry threads.

## Notes

- There are no note-format or vault-data migrations in this release.
