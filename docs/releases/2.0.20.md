# Release Highlights

This release makes pasted long URLs readable inside SideNote2 comments by storing them as shortened markdown links while preserving the full destination.

## Highlights

- Long bare URLs pasted into new or edited side notes are now saved as shortened markdown links, so stored comments stay compact without breaking the underlying link target.
- Existing comments that still contain raw long URLs now render with shortened link labels in the sidebar, even before they are re-saved.
- Page-note previews in `SideNote2 index.md` now show clean link labels instead of raw markdown-link syntax.

## Notes

- Existing stored raw URLs are rewritten into the new shortened markdown-link form the next time that comment is edited and saved.
