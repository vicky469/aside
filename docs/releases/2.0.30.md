# Release Highlights

This release removes Aside's PDF-note support and cleans up the sidebar so unsupported files no longer inherit stale note-local UI.

## Highlights

- Side notes are now markdown-only. PDF note support and the old attachment-comment storage path have been removed from the plugin and docs.
- The note-local sidebar now clears correctly on unsupported files, so switching to a PDF no longer leaves the previous markdown page-note cards visible.
- Sidebar target resolution now treats an unsupported active file as no sidebar target instead of falling back to the last supported note.

## Notes

- Existing Aside interactions continue to work in markdown notes and in the Aside index note.
